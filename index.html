Frequently Asked Questions
Is NexLoad completely free? +
Yes, NexLoad is 100% free. It runs on your own local machine using open-source tools (yt-dlp and ffmpeg), so there are no subscriptions, no ads shown to you by us, and no hidden fees.
Which sites are supported? +
NexLoad supports 1000+ sites including YouTube, TikTok, Instagram, Twitter/X, Facebook, Vimeo, Reddit, Twitch, Dailymotion, SoundCloud, and any other site supported by yt-dlp.
How do I download MP3 audio only? +
Toggle the Format switch to MP3 before clicking Download. In Best mode, yt-dlp will extract and convert the audio to high-quality MP3 using ffmpeg. In Direct and Other modes it finds the best audio stream available.
Why does the server need to be running? +
NexLoad's backend (server.js) runs on your local PC and does the heavy lifting — invoking yt-dlp, ffmpeg, and handling downloads. The frontend on Vercel connects to it via an ngrok tunnel. Without the server running, the frontend can't download anything.
Which mode should I use? +
Use Best when you want maximum video quality (1080p+). Use Direct when you need speed and the file size doesn't have to be optimal. Use Other if the first two methods fail or for obscure platforms.
Is downloading videos legal? +
Downloading for personal offline use is accepted in many jurisdictions. Always respect the content creator's rights and the platform's terms of service. NexLoad is a tool — responsibility for how you use it lies with you.
